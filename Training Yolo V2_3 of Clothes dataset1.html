B. How to Train with [Clothes] dataset

B-A. 產生bonding box & label

${TOOL_ROOT}: /home/e200/Downloads/ BBox-Label-Tool-multi-class

  1. 將圖片放入 ${TOOL_ROOT}/Images/0001/，產生的 Label 檔會在 ${TOOL_ROOT}/Labels/0001
  2. 執行方式：python ${TOOL_ROOT}/main.py，開啟在上方 Image Dir 輸入「 1」，然後按 Load
  3. 右上角可以選取 Class，0:Dark_ST, 1:Light_ST, 2:Dark_LT, 3:Light, 4.Dark_SU, 5.Light_SU, 6. Dark_LU, 7.Dark_LU（可以在${TOOL_ROOT}/class.txt 修改），選完要「按 Confirm Class 才算有選到」
  4. 滑鼠點一下開始框 BBOX，在點一下框選完成，在框的時候可以按 ESC 取消
  5. 右邊選取 BBOX 後按 Delete 可以刪除，Clear All 是全刪
  6. 框 BBox 不用怕框超出界外，有設定超過的會自動修正至邊界
  7. 按下 Next 之後會作儲存的動作輸出 Label 檔案，並出現下一張圖片，框好的圖片會「自動刪除」，這是避免重複產生 Label 檔案，所以 Prev 這個按鍵沒有用（ Next 的鍵盤快捷鍵是「 d」，善用可以提高效率）
  8. 如果這張圖片沒有任何要標的物體，直接 Next 就不會產生 Label 檔，這是避免有空白檔案
  9. 所有資訊可以在 Terminal 上看到，包含當前圖片、框取的 BBox，如果誤刪還可以找得回來
  10. 做到最後一張時按下 Next 會沒反應，並在 Terminal 上看到錯誤訊息，代表已經標完所有圖片
  11. 標記完可以透過 Label 的檔名確認有哪些圖片需要做 training（沒物體或不需要的不會產生Label 檔案）
  • 絕對不要忘記要按 Confirm Class，可以在右邊看到框好的 BBOX 做確認 Class 有沒有選錯
  • 如果圖片太大會無法完整顯示，可以修改 main.py 內的 SIZE_FACTOR 調整圖片放大縮小
  • 要根據圖片的副檔名修改 main.py 內 IMG_TYPE，一次只能讀一種（ .jpg, .png…）

B-B. 更改資訊

  1. 圖片集資料夾名稱改JPEGImages，Label集資料及名稱改labels
  2. 放入${YOLO_ROOT}/clothes內

B-C. 建立Train list 以及 valid list(9:1)

  1. 更改SRC_DIR為JPEGImages路徑，train為輸出train_clothes_v1.txt的檔案名稱，vaild為輸出valid_clothes_v1.txt的檔案名稱
  2. 路徑：${YOLO_ROOT}/Final_Tool/Helpful_Tools/GenTrainList.py
  3. 執行
$ cd darknet_modified/Final_Tool/Helpful_Tools
$ python GenTrainList.py
  1. 產生train_list.txt檔和valid_list.txt檔在darknet_modified內

B-D. 檔案設定

  1. ${YOLO_ROOT}/cfg/action.name改分類內容，移至clothes資料夾，名稱改為 clothes.name
  2. ${YOLO_ROOT}/cfg/action.data 改classes的數量移至clothes資料夾，train_clothes.txt檔路徑，valid_clothes.txt檔路徑，name為clothes.name路徑
  3. yolo-action.cfg 改237行 filter 為(calss 數量+5)*5，244行 class 為 calss 數量

B-E. 轉換為Yolo可以讀的格式(all img sour addr list & ans label list )

  1. 路徑：${YOLO_ROOT}/Final_Tool/Helpful_Tools/TransferToYoloOutput.py
  2. IMG_DIR為JPEGImages路徑，LABEL_DIR為labels路徑，DST_DIR為輸出的路徑，GT_NAME為GT_trainlist_bottle_1004，TEST_NAME為train_list.txt的名稱，Class為個別名稱
  3. 執行
$ cd darknet_modified/Final_Tool/Helpful_Tools
$ python TransferToYoloOutput.py

B-F. 進行train

  1. 執行
$ cd darknet_modified
$ ./darknet detector train clothes/clothes.data clothes/yolo-clothes.cfg tiny-yolo-voc.conv.14
  1. Train 完 weight 在 backup裡/or at {clothes_ROOT}

B-G. 進行test

  1. 執行
$ cd darknet_modified
$ ./darknet detector test clothes/clothes.data clothes/yolo-clothes.cfg backup/yolo-clothes_final.weights clothes/testlist_clothes.txt
  1. 產生predict.txt在${YOLO_ROOT}內
  2. 因為資料使用 append 在 predict.txt 中，需先將之前的 predict.txt 移開

B-H. 計算Accuaracy

  1. 規則：ComputeAccuracy.cpp: 用來計算準確度的程式，需用 g++編譯，結果會顯示於 Terminal 上
    $ ./ComputeAccuracy 「Predict.txt」「 GroundTruth.txt」「Mode」「IoU Threshold」
  2. 執行
$ cd darknet_modified/Final_Tool/Helpful_Tools
$ ./ComputeAccuracy ../../predict.txt ../../clothes/GT_trainlist_clothes.txt 0 0.5

B-I. 執行yolo-darknet with camera

$ cd darknet_modified/clothes
$ ./darknet detector demo clothes/clothes.data clothes/yolo-clothes.cfg backup/yolo-clothes_final.weights

C. How to Train with Pascal voc dataset

C-A. 下載Dataset

  1. 執行
$ wget https://pjreddie.com/media/files/VOCtrainval_11-May-2012.tar
$ wget https://pjreddie.com/media/files/VOCtrainval_06-Nov-2007.tar
$ wget https://pjreddie.com/media/files/VOCtest_06-Nov-2007.tar
$ tar xf VOCtrainval_11-May-2012.tar
$ tar xf VOCtrainval_06-Nov-2007.tar
$ tar xf VOCtest_06-Nov-2007.tar

C-B. 整理數據集

  1. 將所有圖片和成一個資料夾 將所有圖片和成一個資料夾 將所有圖片和成一個資料夾 JPEGImages，所有 label合和成一個資料夾 labels，發現到圖片與 label不完全匹配，將沒對應到label的圖片檔移除
  2. 執行
$ cd darknet_modified/Final_Tool/Helpful_Tools
$ python eliminate_mismatch.py

C-C. 建立Train list 以及 valid list(9:1)

  1. 更改SRC_DIR為JPEGImages路徑，train為輸出train_voc_v1.txt的檔案名稱，vaild為輸出valid_voc_v1.txt的檔案名稱
  2. 路徑：${YOLO_ROOT}/Final_Tool/Helpful_Tools/GenTrainList.py
  3. 執行
$ cd darknet_modified/final_3action/Helpful_Tools
$ python GenTrainList.py
  1. 產生train_list.txt檔和valid_list.txt檔在darknet_modified內

C-D. 檔案設定

  1. ${YOLO_ROOT}/cfg/action.name改分類內容，移至voc_dataset資料夾
  2. ${YOLO_ROOT}/cfg/action.data 改classes的數量移至voc_dataset資料夾，train_list.txt檔路徑，valid_list.txt檔路徑，name為action.name路徑
  3. yolov2-tiny-voc.cfg改118行filter為(calss 數量+5)*5，124行class為calss數量

C-E. 轉換為Yolo可以讀的格式(all img sour addr list & ans label list )

  1. 路徑：${YOLO_ROOT}/Final_Tool/Helpful_Tools/TransferToYoloOutput.py
  2. IMG_DIR 為 JPEGImages 路徑，LABEL_DIR 為 labels 路徑，DST_DIR 為輸出的路徑，GT_NAME 為 GT_trainlist_voc.txt，TEST_NAME 為 train_list.txt 的名稱，Class 為個別名稱
  3. 執行
$ cd darknet_modified/Final_Tool/Helpful_Tools
$ python TransferToYoloOutput.py

C-F. 建立pre-trained weight

  1. 執行
$ cd darknet_modified
$ ./darknet partial voc_dataset/yolov2-tiny-voc.cfg yolov2-tiny-voc.weights yolov2-tiny-voc.conv.13 13

C-G. 進行train

  1. 執行
$ cd darknet_modified
$ ./darknet detector train voc_dataset/voc.data voc_dataset/voc.cfg yolov2-tiny-voc.conv.13
  1. Train 完 weight 在 backup裡

C-H. 進行test

  1. 執行
$ cd darknet_modified
$ ./darknet detector test voc_dataset/voc.data backup/yolov2-tiny-voc_final.weight voc_dataset/GT_trainlist_bottle_1004.txt
  1. 產生predict.txt在${YOLO_ROOT}內
  2. 因為資料使用 append 在 predict.txt 中，需先將之前的 predict.txt 移開

C-I. 計算Accuaracy

  1. 規則：ComputeAccuracy.cpp: 用來計算準確度的程式，需用 g++編譯，結果會顯示於 Terminal 上
    $ ./ComputeAccuracy 「Predict.txt」「 GroundTruth.txt」「Mode」「IoU Threshold」
  2. 執行
$ cd darknet_modified/final_3action/Helpful_Tools
$ ./ComputeAccuracy ../../predict.txt ../../voc_dataset/GT_trainlist_voc.txt 0 0.5
